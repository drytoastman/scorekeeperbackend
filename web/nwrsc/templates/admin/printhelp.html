General printing setup for blank 5x8 index cards (office depot, staples, etc). At least what has worked for me.

Very first time (Windows):

  • Open your printers folder
  • Select printer properties for desired printer
  • Click "Printing Preferences"
  • Find the paper source section
  • Create a custom paper source of size 5" width x 8" length
  • Name it indexcard or what ever you like

Print cards

  • Download the PDF file of cards (pick the ordering you want)
  • Open in Acrobat Reader and Click print
  • In the preferences dialog:
    • Set Page Scaling to "None"
    • Select "Choose Paper Source by PDF page size"
  • Print

Note that you want to feed in index cards in length wise just like regular paper. I would test print just one page (one card) to start with to see if things work correctly
