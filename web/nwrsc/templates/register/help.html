There are four selections in the menu bar for the regisration site.
  1. Series Selection
    This is a quick menu selection for switching between the active series for the year
  2. Profile
    This is where you can view and modify the information for your profile that all series on the site will have access to. In general:
    • Name is important for all events
    • Email is required to reset username/password and to get any event notices from the event staff
    • Do Not Contact specifies that any group emails sent from the event staff will not be sent to you
    • Barcode is needed for any series that are using barcodes for staging
    • Address and contact information are required by some series onsite but not others

    If a series series requests that you have a barcode set in your profile but you don't have one, you will see a warning at the top. If you already have a barcode on your helmet or vehicle, please go to the Profile page, edit profile, and enter the barcode value there. If you do not have a barcode yet, one will be provided if necessary at the event and you can ignore any warnings at the top.

  3. Cars
    This is where you create/edit/delete cars that will be used just for the selected series. A car contains not only the description but number, class and index information.
    • Numbers in a class have to be unique to a driver for the series, if someone else is already using a number, you will be required to select a different number.
    • Classing is based on the local series and how they divide cars based on their performance abilities.
    • Index is only needed for classes that group multiple classes of cars into one. Selecting an index will only be available if required.

  4. Events
    This is where you register cars from the above stuff for the upcoming events in the series
    • Clicking on the +/- icon or event name will show/hide the events of the selected series
    • The Left side show information about the event such as when registration closes, number of entries, etc.
    • The Add/Change button will open a dialog that lets you select which cars (from above) to registers
    • If the event is taking online payments and you have registered a car, a Make Payment button will display which takes you to either a Paypal or a Square payment site
